# Base de dados Regiões de Saúde ANS

Arquivo CSV com a lista de **Regiões de Saúde do Brasil** por município, baseado nos dados do anexo da Resolução Normativa ANS Nº 665, de 26 de fevereiro de 2026.

## Colunas

| Coluna | Descrição |
|--------|-----------|
| `UF` | Sigla do estado |
| `NOME DA REGIAO DE SAUDE` | Nome da Região de Saúde |
| `COD. DA REGIAO DE SAUDE` | Código de 5 dígitos da Região de Saúde |
| `NOME DO MUNICIPIO` | Nome do município |
| `COD. IBGE DO MUNICIPIO` | Código IBGE de 6 dígitos do município |

**5.554 registros** — todos os municípios brasileiros dos 26 estados + DF.
